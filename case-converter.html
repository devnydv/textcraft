Common questions

What is a text case converter?
A case converter transforms text between different capitalization formats. Writers use it for consistent formatting; developers use it to convert variable names, API keys, and identifiers between naming conventions like camelCase, snake_case, and PascalCase.
What is camelCase and when is it used?
camelCase writes each word with no spaces, capitalizing every word except the first (e.g. helloWorld). It's the standard naming convention for variables and functions in JavaScript, Java, and many other languages.
What is the difference between snake_case and kebab-case?
snake_case uses underscores to separate words (hello_world) and is common in Python and database column names. kebab-case uses hyphens (hello-world) and is typically used in URLs, CSS class names, and HTML attributes.
What is Title Case?
Title Case capitalizes the first letter of every major word. It's used in headings, article titles, book names, and product names to give text a formal, polished appearance.
Is there a character limit?
No — the converter handles any amount of text entirely in your browser. Paste an entire document if you need to.
