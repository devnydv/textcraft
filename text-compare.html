How the Text Compare tool works

A line-by-line and word-by-word diff engine that shows exactly what changed between any two texts.

⇌ Side-by-side View

See original and modified text in two aligned columns, with each line colour-coded so differences are immediately obvious.

≡ Unified View

View all added and removed lines in a single column, similar to a git diff output — ideal for quick scanning.

🔤 Word-level Highlights

Inside changed lines, individual words that were added or removed are highlighted with their own colour so you never miss a detail.

🔡 Ignore Case

When enabled, "Hello" and "hello" are treated as identical. Useful when comparing text that may have different capitalisation.

⎵ Ignore Whitespace

Collapses multiple spaces before comparing, so formatting differences don't show up as meaningful changes in the diff.

📊 Similarity Score

The stats bar shows an overall similarity percentage so you can instantly gauge how close the two texts are before reading the full diff.

Frequently Asked Questions

Is the Text Compare tool free?▾

Yes, completely free with no account required. All TextCraft tools are free to use without limits.

Is my text private?▾

Yes. The entire diff runs in your browser using JavaScript. Neither text is ever sent to a server, stored, or logged. Your content stays on your device.

What is the difference between Split and Unified view?▾

Split view shows both texts side by side in two columns — ideal for reading context. Unified view shows a single column where added lines are marked with + and removed lines with −, similar to a git diff. Use unified view for a compact overview.

How does word-level highlighting work?▾

For lines that are similar but not identical (marked as "changed"), the tool runs a second word-level diff and highlights the exact words that were added or removed within that line — so you can spot a single word change in a long sentence instantly.

Is there a text size limit?▾

There is no hard limit. The diff algorithm runs locally, so capacity depends on your device. Documents of thousands of lines compare in milliseconds on modern hardware.
